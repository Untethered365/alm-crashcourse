# The ALM workshop you’ve been waiting for: Azure DevOps made easy!

This repository holds the content, labs, instructions, samples and other learning materials associated with the introductory Power Platform ALM workshop delivered by [Sean Astrakhan](https://www.linkedin.com/in/sean-astrakhan/) and [Parvez Ghumra](https://www.linkedin.com/in/parvezghumra/). Don't hesitate to reach out to us directly via LinkedIn in case of any queries.

We deliver this as an all-day hands-on workshop to delegates at major conferences around the world. This provides an interactive learning experience, with dedicated access to environments, Azure DevOps/Github organizations, source control repositories, agents/runners etc, as well as plenty of opportunities to ask question. So far it has been delivered at the following events and is planned to be delivered at more events over the course of the coming months. If you would like us to bring this workshop to your event, feel free to reach out.
